工作流程

  • Workspace：工作区
  • Index / Stage：暂存区
  • Repository：本地仓库
  • Remote：远程仓库

文件状态

参数配置

git config

  • git配置文件位于～/.gitconfig
  • --global为全局配置，不加此选项只对当前项目生效。

新建仓库 git init | git clone

文件状态 git status

  • -s 紧凑（short）

提交信息 git log

操作记录 git reflog

新增文件 git add

删除文件 git rm

  • -f 用于已修改的状态
  • –cached 用于只在Git版本库中删除，工作目录中保留。

提交文件 git commit

推送 git push

  • -u 关联远程分支
  • -f (force) 强制覆盖

比较文件 git diff

  • 默认是未暂存的改动
  • –cached 已暂存的改动
  • HEAD 未暂存 + 已暂存
  • –stat 摘要（statistics）

时光穿梭

变基 git rebase

  • -i 会列出所有变基的条目以供操作
  • 可在newbase后追加～N，N为条目数量。

重置 git reset

  • –soft 仅在本地仓库移动HEAD repository
  • –mixed 默认实现 repository + index
  • –hard repository + index + workspace

回滚 git revert

  • 新增一个commit来取消一个历史commit的内容，以保存完整的提交记录。
